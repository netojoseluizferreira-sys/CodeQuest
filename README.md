# 🚀 CodeQuest

**Uma plataforma educacional gamificada para o ensino de lógica de programação e Python.** 

## 📋 Sobre o Projeto
O **CodeQuest** surge para resolver a dificuldade de iniciantes em encontrar trilhas estruturadas de aprendizado.  O sistema utiliza uma abordagem guiada por "mundos" progressivos, onde o usuário evolui através de exercícios práticos e elementos de gamificação como XP e níveis. 

## 🛠️ Fase Atual: Protótipo (Terminal)
Atualmente, o projeto valida sua lógica central através de uma interface simplificada: 
* **Lógica:** Desenvolvida em **Python**, focada nas regras de negócio e interação via terminal (input/output). 
* **Armazenamento:** Utiliza arquivos **JSON** para persistência leve de dados como perfis e progresso. 
* **Controle:** Todo o histórico é gerenciado via **Git**. 

## 🕹️ Funcionalidades Disponíveis
* **Perfil do Usuário:** Criação e salvamento local de progresso. 
* **Gamificação:** Sistema de XP e progressão de níveis funcional. 
* **Exercícios:** Resolução de desafios interativos com feedback imediato de acerto ou erro. 

## 🔧 Como Rodar o Protótipo
1. Certifique-se de ter o **Python 3** instalado em sua máquina. 
2. Clone este repositório para o seu computador.
3. No terminal, acesse a pasta do projeto.
4. Execute o arquivo principal:
   ```bash
   python app.py
